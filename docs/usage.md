# Инструкция пользователя

## 1. Скачивание проекта

```bash
git clone https://github.com/USERNAME/security-lab-complex.git
cd security-lab-complex
```

## 2. Запуск

```bash
sudo ./start.sh
```

## 3. Использование меню

После запуска пользователь выбирает нужный пункт меню. Например:

- пункт 1 — аудит системы;
- пункт 4 — поиск уязвимых настроек;
- пункт 6 — защита SSH;
- пункт 7 — настройка Firewall;
- пункт 8 — защита от атак (Fail2Ban);
- пункт 11 — сформировать отчёт.

## 5. Где смотреть результаты

- журналы работы программы: `logs/actions.log`;
- отчёты проверки: `reports/`;
- резервные копии: `backups/`.
