# Security Lab Complex

## Overview

Security Lab Complex — модульный программный комплекс для аудита, анализа и базового усиления безопасности Linux-систем.

Проект разработан как единая платформа для автоматизации типовых задач по обеспечению информационной безопасности операционных систем семейства Linux. Комплекс объединяет инструменты аудита, контроля конфигурации, мониторинга и настройки механизмов защиты в рамках единого терминального интерфейса.

Основная идея проекта заключается в сокращении количества ручных операций при настройке безопасности системы и предоставлении централизованного инструмента для проведения проверок и применения защитных мер.

Решение может использоваться как в образовательной среде для изучения технологий защиты Linux, так и в качестве вспомогательного инструмента для базовой подготовки тестовых и учебных серверов.

---

## Key Features

### Security Auditing

* Анализ состояния операционной системы
* Проверка пользователей и привилегий
* Анализ открытых сетевых портов
* Выявление потенциально небезопасных конфигураций
* Расчёт итоговой оценки защищённости системы (Hardening Score)

### System Hardening

* Защита SSH-сервера
* Настройка межсетевого экрана UFW
* Конфигурация Fail2Ban
* Усиление параметров ядра Linux
* Настройка политики паролей
* Автоматические обновления безопасности
* Контроль системных служб

### Monitoring & Compliance

* Мониторинг журналов безопасности
* Настройка Auditd
* Контроль целостности файлов
* AppArmor / SELinux
* Анализ изменений критически важных файлов

### Reporting

* Автоматическое формирование отчётов
* Журналирование действий пользователя
* Сохранение результатов аудита
* Документирование выполненных настроек

## Architecture

Проект построен на основе модульной архитектуры.

Каждый функциональный блок реализован в виде отдельного Bash-модуля, отвечающего за конкретную область обеспечения безопасности.

Такой подход обеспечивает:

* независимость компонентов;
* простоту сопровождения;
* возможность масштабирования;
* удобство тестирования;
* лёгкое добавление новых модулей.

Центральной точкой управления выступает панель Security Lab Complex, через которую пользователь получает доступ ко всем функциям системы.

Основные цели проекта:

* снижение количества ошибок ручного конфигурирования;
* создание единой среды управления защитой системы;
* демонстрация практических методов hardening Linux;
* поддержка учебного процесса по дисциплинам информационной безопасности.

## Project Structure

* **Audit Modules** — анализ системы и оценка защищённости
* **Hardening Modules** — настройка защитных механизмов
* **Monitoring Modules** — аудит и журналирование
* **Reporting Modules** — формирование отчётов
* **Configuration Layer** — централизованное управление параметрами
