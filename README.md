# Security Lab Complex

**Security Lab Complex** — интерактивная терминальная панель для автоматизированного аудита и настройки безопасности Linux-систем.

Проект объединяет **19 модулей**, покрывающих все основные аспекты защиты ОС: от анализа системы и проверки пользователей до настройки ядра, межсетевого экрана, системы аудита и изоляции приложений. Подходит для учебных лабораторных работ и быстрой базовой настройки серверов.

## Возможности

| Область | Модули |
|---------|--------|
| 🔍 Аудит | система, пользователи, порты, оценка безопасности |
| 🛡 Защита | права доступа, SSH, UFW, Fail2Ban, sysctl, пароли, службы |
| 📦 Доп. защита | AppArmor, auditd, Firejail, автообновления, целостность файлов |
| 📊 Отчёты | полный аудит, формирование отчёта |

## Запуск

```bash
git clone https://github.com/ErgashevBehzod/security-lab-complex.git
cd security-lab-complex
sudo ./start.sh
```

Для режима только аудит (без прав root):
```bash
./start.sh
```

## Требования

- Debian, Ubuntu, Kali Linux или совместимый дистрибутив
- Bash
- Права root для выполнения настроек (не обязательно для аудита)
- Доступ к `apt`

## Конфигурация

Все параметры в `config/default.conf`:

```conf
FAIL2BAN_BANTIME="3600"
FAIL2BAN_FINDTIME="600"
FAIL2BAN_MAXRETRY="5"
PASS_MAX_DAYS="90"
PASS_MIN_DAYS="7"
PASS_WARN_AGE="14"
PASS_MIN_LEN="12"
```

## Анализ аналогов

| Продукт | Недостаток |
|---------|-----------|
| Lynis | только аудит, нет настройки |
| CFEngine | сложный, корпоративный |
| FireHOL / Shorewall | только firewall |
| Fail2Ban | только защита от брутфорса |
| OpenVAS | только сканирование |
| GitHub-скрипты | нет единой структуры |

**Security Lab Complex** — единственное комплексное решение, объединяющее аудит, настройку, отчёт и обучение в одной терминальной панели.

## Важно

Проект предназначен для учебных и лабораторных целей. Перед применением на реальном сервере проверьте настройки.
